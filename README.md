# 📍 Smart Campus Navigation System

## 📌 Problem
Visitors and new students face difficulty navigating large campuses due to lack of internal navigation systems. They often struggle to find buildings, classrooms, and faculty cabins.

---

## 💡 Solution
A smart web-based campus navigation system that allows users to search for locations, visualize routes, and dynamically navigate within the campus.

---

## 🚀 Features

### 🔍 Navigation Features
- Search campus locations (Library, Admin, etc.)
- Dynamic route visualization between locations
- Route updates based on **current position (not static)**
- Return navigation to **Main Gate from anywhere**

### 📍 Interactive Map
- "You are here" animated user indicator
- Interactive location markers (clickable)
- Responsive map using percentage-based positioning

### 🖼️ Smart UI Enhancements
- Location labels on map
- Image popup when clicking buildings
- Smooth animated movement of user dot

### 👨‍🏫 Faculty Finder
- Search faculty by name
- Displays cabin number and details
- Helps students locate professors easily

---

## 🛠 Tech Stack
- HTML (Structure)
- CSS (UI Design & Animations)
- JavaScript (Logic & Interactivity)

---

## 🎯 How It Works
1. User searches a location (e.g., Library)
2. System calculates route from current position
3. Displays path using dynamic line rendering
4. Moves user indicator smoothly to destination
5. Updates current position for next navigation
6. Users can return to **Gate anytime**

---

## 💡 Unique Features
- 🔄 Dynamic routing (not fixed start point)
- 🚪 Gate navigation from any location
- 🧭 Simulated real-time navigation
- 🖼️ Visual building preview (image popup)
- 👨‍🏫 Faculty cabin lookup system
