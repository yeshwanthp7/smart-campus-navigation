# Smart Campus Navigation System

## 📌 Problem
Visitors and new students face difficulty navigating large campuses.

## 💡 Solution
A smart web-based navigation system that allows users to search locations and view routes inside campus.

## 🚀 Features
- Search campus locations
- Visual route display
- "You are here" indicator
- Interactive map with markers

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
